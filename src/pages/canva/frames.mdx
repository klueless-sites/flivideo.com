## Technique 1

- Click elements
- Click on the 'Frames' icon
- Select a frame

## Technique 2

- Type 'frame + keyword' in the search bar
- Select a frame

## Technique 3

- Click on the '...' icon for a specific frame
- Click on the keyword
- Select a frame
