## Use the zoom controls

- Use the zoom controls in the bottom right corner

## Use keyboard shortcuts

- Use the 'Command + +' key to zoom in
- Use the 'Command + -' key to zoom out

## Use mouse shortcuts

- Use the 'Ctrl + Zoom' key to zoom in/out
