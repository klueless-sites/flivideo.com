## Use the `Duplicate` icon

- Select image
- Click on the `Duplicate` icon

## Use keyboard shortcut

- Select image
- Press the `Command+D` key (my favorite)
- Press the `Ctrl+D` key (my favorite)

## Use mouse shortcut

- Select image
- Click `Option + Drag` to duplicate the image
- Click `Alt + Drag` to duplicate the image
