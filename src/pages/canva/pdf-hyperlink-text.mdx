## Using keyboard shortcuts

- Select text for hyperlink
- Press the `Command + K` key

## Using the UI controls

- Right click on the text and select `Link`
- Enter the URL
- Click `Apply`

## Save the document as a PDF

- Save the document as a PDF
- Open the PDF
- Click on the hyperlink
