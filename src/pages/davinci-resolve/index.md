## Keyboard Shortcuts

### Cut Screen

List of useful options when working in the 'Cut' screen.


- `Option + /` - Play from 'in' to 'out' on a selected clip
- `q` - Switch between 'Source Clip' or 'Timeline' view
- `i` - Set mark 'in' point
- `o` - Set mark 'out' point
- `Opt+X` - Clear mark in/out
- `Shift+F12` - Append source clip to timeline 
- `/` - Play around the current selection, you need to click between to selections for this to work


#### Playback

- `Space` - Play/Pause
- `J` - Play backwards
- `L` - Play forwards
- `Shift+J` - Play backwards fast
- `Shift+L` - Play forwards fast
- `Command + /` - Play again
- `Fn+Left arrow` - Go to start of timeline
- `Fn+Right arrow` - Go to end of timeline
- `Up arrow` - Go to previous clip or edit
- `Down arrow` - Go to next clip or edit
- `Left arrow` - Go to previous frame
- `Right arrow` - Go to next frame
